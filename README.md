## Опис


У рамках лабораторної роботи реалізовано алгоритмічну задачу — **гру «Життя» (Conway’s Game of Life)**  
Поле є **тором** (замкнене по горизонталі та вертикалі).  
Жива клітина позначається `x`, мертва — `.`.

Програма:

- читає кількість поколінь;
- розміри поля;
- початковий стан поля;
- обчислює розвиток гри на вказану кількість поколінь;
- виводить фінальний стан поля.

Також виконано:

- налаштування GitHub Actions для автоматичного запуску тестів;
- створено PR із успішним тестом;
- створено PR із тестом, що падає;

## Вимоги

- Python 3.11+
- `pip`
- `pytest`

## Налаштування середовища / встановлення залежностей

Команда для встановлення залежностей:

```bash
pip install -r requirements.txt
```

## Запуск тестів

Команда для запуску тестів:

```bash
python -m pytest
```

## Запуск програми вручну

Вхідний файл input.txt:

3
8 5
........
..x.....
..x.....
..x.....
........

Команда для запуску тестів у Windows (CMD, а не PowerShell):

```bash
python lab1.py < input.txt
```

Команда для запуску тестів у PowerShell:

```bash
gc input.txt | python lab1.py
```

Команда для запуску тестів у Linux:

```bash
python3 lab1.py < input.txt
```
